ValutoBot

Telegram-бот для конвертации валют

Python-бот, использующий API Telebot и данные Центрального банка Российской Федерации. Он конвертирует валюты по текущему обменному курсу. Пользователь вводит запрос в формате XXXYYY (например, USDRUB 100) и получает мгновенный результат.

Технологии: Python, Telebot, Requests, Central Bank of the Russian Federation API.

MyNotes

Telegram-бот для управления личными заметками

Python-бот для создания, хранения и организации заметок через Telegram. Пользователь может создавать заметки с названием и текстом, просматривать все сохранённые записи, редактировать содержимое и удалять ненужные заметки. Все данные надёжно хранятся в PostgreSQL базе данных с привязкой к ID пользователя.

Технологии: Python, Telebot, PostgreSQL (psycopg), inline-клавиатуры для навигации.
